{% blocktrans %}1. Foydalanish Shartlari{% endblocktrans %}

{% blocktrans %}

Ushbu shartlar dgdconsulting.uz xizmatlaridan foydalanish qoidalarini belgilaydi. Saytimizga kirish orqali ushbu shartlarga rioya qilishga rozilik bildirasiz.

Xizmatlar ko‘rsatish

{% endblocktrans %}
  • {% blocktrans %}Saytimiz faqat qonuniy maqsadlarda foydalanish uchun mo‘ljallangan{% endblocktrans %}
  • {% blocktrans %}Foydalanuvchilar xizmatdan noqonuniy yoki noto‘g‘ri maqsadlarda foydalanmasliklari shart.{% endblocktrans %}
  • {% blocktrans %}Kompaniya xizmatni istalgan vaqtda o‘zgartirish yoki to‘xtatib qo‘yish huquqiga ega.{% endblocktrans %}

{% blocktrans %}Javobgarlik{% endblocktrans %}

  • {% blocktrans %}Saytda mavjud ma’lumotlar faqat umumiy ma’lumot berish uchun mo‘ljallangan.{% endblocktrans %}
  • {% blocktrans %}Biz texnik xatoliklar yoki saytga kirishdagi uzilishlar uchun javobgar emasmiz.{% endblocktrans %}
